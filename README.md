# Persona

O aplicativo foi projetado principalmente para estudantes universitários que dividem apartamento ou moram em repúblicas.

O foco é ajudar usuários que precisam dividir contas de forma rápida, simples e organizada, evitando conflitos relacionados a dinheiro e cobranças.

---

# Heurística de Nielsen em foco

A heurística priorizada foi:

## Visibilidade do Status do Sistema

O protótipo foi pensado para mostrar claramente:

* Quem deve
* Quem recebeu
* Valores pendentes
* Status dos pagamentos

Foram utilizados destaques visuais, organização em cards e informações resumidas para facilitar a compreensão do usuário.

# Fluxo de Tarefa

## Cadastro e divisão de uma conta de luz entre 3 pessoas

1. O usuário acessa o Dashboard principal.
2. Seleciona o grupo “Casa”.
3. Clica no botão “Adicionar Gasto”.
4. Insere:

   * Nome da conta
   * Valor
   * Categoria
5. Seleciona os 3 participantes.
6. Escolhe a divisão igualitária.
7. Confirma o cadastro da despesa.
8. O sistema atualiza automaticamente os valores de quem deve para quem.
9. Os participantes recebem a informação da nova cobrança no extrato de dívidas.
